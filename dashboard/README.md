# PAA 2027 Rising Senior Draft Dashboard

Interactive Next.js dashboard backed by the latest rising-senior draft board.

```bash
npm install
npm run data
npm run dev
```

The dashboard is a standard Next.js application and can be imported directly
into Vercel with `dashboard` selected as the project root. No environment
variables or external services are required.

Run `npm run data` after regenerating the model outputs to refresh the dashboard
JSON, then run `npm run build` before deployment.
